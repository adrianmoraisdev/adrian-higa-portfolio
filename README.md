# Adrian Higa Portfolio

Landing page profissional para apresentar habilidades em .NET back-end, IA aplicada, RPA, APIs REST e integracao de sistemas.

## Publicacao

Este projeto e estatico. Pode ser publicado diretamente pelo GitHub Pages usando a branch `main` e a pasta raiz.

## Estrutura

- `index.html`: conteudo principal da landing page
- `styles.css`: design responsivo e identidade visual
- `script.js`: animacao de particulas no fundo
- `assets/`: imagem visual da arquitetura neural
